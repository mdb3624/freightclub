Generic pill badge for status indicators, counts, and tags. Supports semantic variants aligned with FreightClub's status color system.

```jsx
<Badge variant="success">Delivered</Badge>
<Badge variant="warning" dot>At Risk</Badge>
<Badge variant="error">Overdue</Badge>
<Badge variant="rpm_high">$2.45/mi</Badge>
<Badge variant="bronze">Quick Pay</Badge>
```

Use `StatusBadge` for load statuses. Use `Badge` for everything else — payment terms, equipment types, performance tiers.
