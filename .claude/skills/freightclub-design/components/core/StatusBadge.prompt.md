Maps FreightClub load lifecycle statuses to colored pills. Use only for the API status field on loads.

```jsx
<StatusBadge status="DRAFT" />
<StatusBadge status="OPEN" />
<StatusBadge status="CLAIMED" />
<StatusBadge status="IN_TRANSIT" />
<StatusBadge status="DELIVERED" />
<StatusBadge status="SETTLED" />
<StatusBadge status="CANCELLED" />
```

Statuses: DRAFT (grey) · OPEN (blue) · CLAIMED (amber) · IN_TRANSIT (purple) · DELIVERED (green) · SETTLED (teal) · CANCELLED (red)
