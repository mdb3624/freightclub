# Shipper / Administrator UI Kit

**Persona:** Shipper (desktop office manager, logistics coordinator)  
**Theme:** "Classic Cream & Metallic Bronze"  
**Viewport:** 1280px desktop

## Screens Covered
- `index.html` — Shipper Dashboard: KPI bar, shipment status table, action zone

## Design Language
- Canvas: `#EFEBE0` warm cream
- Panels: `#FFFFFF` + `1px solid #D0D0D0` border + `8px` radius + subtle shadow
- Primary CTA: Bronze gradient (`#C9A46A → #B08D57 → #8C6D3F`) with tactile inset shadows
- Typography: Sora (headings) + Inter (body)
- Table rows: 48px height, `#E8E3D8` dividers, hover → `#F8F9FB`
- Form inputs: 40px height, 4px radius, 2px bronze focus ring

## Key Components Used
- `StatusBadge` — load lifecycle states
- `Avatar` — shipper profile circle with bronze ring
- KPI cards with Sora 48px heavy numbers
- Recessed bronze progress bars for shipment transit
- Quick Action Panel with full-width bronze CTAs
- Carrier Feed with avatars + star ratings
