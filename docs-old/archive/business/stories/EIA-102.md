# EIA-102 — Regional Diesel Price Market Ticker

**As a** Driver / Owner-Operator,
**I want** to see current diesel prices for all five U.S. regions in the market ticker bar on every screen,
**so that** I can factor accurate fuel cost into load decisions without leaving the app.

## Acceptance Criteria
- [ ] Ticker displays all five entries in the exact format `DIESEL EAST: $X.XX/gal`, `DIESEL MIDWEST: $X.XX/gal`, `DIESEL SOUTH: $X.XX/gal`, `DIESEL ROCKIES: $X.XX/gal`, `DIESEL WEST: $X.XX/gal`, scrolling continuously — satisfies **FR-06**
- [ ] All price values are formatted as USD per gallon rounded to exactly two decimal places (e.g. `$3.72`, not `$3.7215`) — satisfies **FR-07**
- [ ] Core page layout is visible within 1s; ticker data populates asynchronously within 3s of page load — satisfies **NFR-03**; ticker rendering does not block initial paint
- [ ] If the backend cache is empty and the EIA API is unreachable, each region displays `--` (e.g. `DIESEL WEST: --`) rather than an error or blank — satisfies **NFR-05**
- [ ] If cached data is older than 48 hours, a stale-data visual indicator is rendered alongside the ticker values — satisfies **FR-09**
- [ ] **Security / Attribution:** The `FuelPriceTicker` component fetches exclusively from `/api/v1/market/fuel-prices` (backend proxy); no client-side requests to `api.eia.gov` are made — satisfies **NFR-08**; a `"Data: U.S. EIA"` attribution label is present in the app's data sources or about section — satisfies **NFR-10**
- [ ] Cyclomatic complexity ≤ 10 in `FuelPriceTicker` React component — satisfies reviewer checklist quality gate
- [ ] Ticker and color-coded delta indicators render correctly in both light and dark mode — satisfies **FR-08** (color rendering; delta indicators delivered by EIA-103)
