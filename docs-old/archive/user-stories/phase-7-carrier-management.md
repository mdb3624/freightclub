# User Stories — Phase 7: Advanced Carrier Management

## Trucker Profile Enhancements

- **As a trucker**, I want to add my truck and trailer details (type, dimensions, capacity) to my profile so that shippers can verify I have the right equipment before assigning me a load.
- **As a trucker**, I want to set my preferred lanes (origin and destination regions) on my profile so that the platform can surface relevant loads and shippers can find me for repeat routes.
- **As a trucker**, I want to set my availability (days and hours I'm available for pickup) so that shippers know when they can count on me.

## Suggested Loads

- **As a trucker**, I want to see suggested loads that match my preferred lanes and equipment type so that I spend less time filtering and more time running freight.
- **As a trucker**, I want to filter the load board by weight range and minimum pay rate so that I can exclude loads that don't meet my operational requirements.
- **As a trucker**, I want to see how many truckers have viewed or expressed interest in a load so that I can gauge competition and act quickly on high-demand loads.

## Shipper Tools

- **As a shipper**, I want to view a trucker's public profile (rating, reviews, equipment, completed load history) before assigning them a load so that I can make an informed carrier selection.
- **As a shipper**, I want to maintain a preferred carrier list of trusted truckers so that I can quickly re-use carriers who have performed well on my lanes.
- **As a shipper**, I want to directly assign a load to a trucker on my preferred carrier list so that reliable carriers get first access to my freight without competing on the open board.
- **As a shipper**, I want to block a carrier so that a trucker who has performed poorly cannot claim any of my future loads.

## Load Posting Guidance

- **As a shipper**, I want the load creation form to surface validation tips and prompts (accurate weight, state special requirements, competitive rates, realistic windows) so that I post loads that attract quality carriers quickly.

## Pickup Urgency

- **As a trucker**, I want to see a pickup urgency signal on load cards (e.g. "Picks up tomorrow"; amber highlight if < 24 hr) so that I can prioritize time-sensitive loads before they expire.

## Load Board Sorting

- **As a trucker**, I want to sort the load board by RPM, distance, or pickup date so that I can rank loads by the criteria most relevant to my current situation.
