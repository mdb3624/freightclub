# S-002 — Default Pickup Location

**As a** shipper,
**I want** to set a default pickup location,
**so that** my origin address pre-fills when I create a new load.

## Acceptance Criteria

- [ ] Profile settings include a "Default Pickup Location" section with street, city, state (2-letter code), and zip fields.
- [ ] When a default pickup location is saved, it pre-fills the origin fields on the Load Creation form.
- [ ] The pre-filled origin can be overridden per load without affecting the saved default.
- [ ] If no default is set, origin fields start blank.
- [ ] Changes to the default do not retroactively alter previously created loads.
