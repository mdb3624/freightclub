# S-010 — Shipper Dashboard Status Summary Strip

**As a** shipper,
**I want** a status summary strip above my loads table showing counts by status,
**so that** I can assess my load portfolio at a glance without scanning every row.

## Acceptance Criteria

- [ ] A summary strip is displayed above the loads table on the shipper dashboard.
- [ ] The strip shows counts for: Open, Claimed, In Transit, and Delivered.
- [ ] Counts reflect the current state of the shipper's loads (scoped to their tenant).
- [ ] Each status tile is clickable and filters the loads table to that status.
- [ ] Counts update in real time (or on page load) without a full page refresh.
- [ ] A count of 0 still displays the tile (shows "0") so the shipper can see all possible states.
- [ ] Cancelled and Settled loads are excluded from the summary strip (or shown in a secondary row).
