# S-024 — Automatic Invoice on Delivery

**As a** shipper,
**I want** an automatic invoice generated on delivery confirmation,
**so that** I don't have to create invoices manually or wait for the trucker to submit one.

## Acceptance Criteria

- [ ] When a load transitions to DELIVERED, an invoice is automatically generated.
- [ ] The invoice includes: shipper and trucker details, load reference, origin/destination, delivery date, agreed pay rate, payment terms, and total amount due.
- [ ] The invoice is accessible from the load detail page for both shipper and trucker.
- [ ] The shipper receives an in-app and email notification with a link to the invoice.
- [ ] The invoice is downloadable as a PDF.
- [ ] Invoice generation is idempotent — re-delivering a load does not create a duplicate invoice.
