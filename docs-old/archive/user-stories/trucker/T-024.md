# T-024 — Email Notification — Load Cancelled by Shipper

**As a** trucker,
**I want** to receive an email when a shipper cancels a load I have claimed,
**so that** I am informed immediately and can pursue other loads.

## Acceptance Criteria

- [ ] When a CLAIMED load transitions to CANCELLED status, a `LoadEvent` row is created with `event_type = CANCELLED`
- [ ] An email is triggered to the trucker's registered email address
- [ ] The email includes load ID, origin, destination, and the shipper's provided reason for cancellation
- [ ] The email subject clearly identifies the action: "Your claimed load [ID] has been cancelled"
- [ ] Email is sent within 5 seconds of the cancellation
- [ ] If email send fails, the cancellation still succeeds and an error is logged
