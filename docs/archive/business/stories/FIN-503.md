# FIN-503 — Payment Processing, Settlement & SETTLED State Transition

**As a** Trucker,
**I want** to receive payment through the platform via ACH or Stripe after my invoice is approved,
**so that** I am paid without leaving the platform, and both parties have a shared, auditable settlement record.

## Acceptance Criteria
- [ ] `PaymentService` (application layer) processes an invoice payment via a pluggable `PaymentGatewayPort` (output port); concrete adapters: `StripePaymentAdapter` and `AchPaymentAdapter`; selected via `tenant_billing_config.payment_method`
- [ ] On successful payment gateway confirmation, `invoices.status` transitions to `'PAID'`; the `LoadAggregate` transitions to `SETTLED` (final state, no further transitions permitted); `LoadSettledEvent` is written to `message_outbox` and `audit_log`
- [ ] `payments` table: `id UUID`, `tenant_id UUID`, `invoice_id UUID`, `load_id UUID`, `gateway_reference VARCHAR`, `amount NUMERIC(10,2)`, `currency_code VARCHAR(3)`, `status VARCHAR`, `paid_at TIMESTAMPTZ`; unique constraint on `(tenant_id, gateway_reference)`; RLS on `tenant_id`; index on `(tenant_id, status)` — satisfies **REQ-103** and **NFR-503**
- [ ] Trucker must register a bank account or Stripe Connect account before any payment can be disbursed; `carrier_payout_accounts` table stores encrypted routing/account details or a `stripe_account_id`; no plaintext bank data in the DB
- [ ] Payment failure (insufficient funds, gateway timeout) sets `invoices.status = 'PAYMENT_FAILED'`; a retry is scheduled via `message_outbox` after 24 hours (max 3 retries); shipper receives a notification on each failure
- [ ] Cyclomatic complexity ≤ 10 in `PaymentService.process()` — satisfies **NFR-501**; gateway-specific logic is fully encapsulated in adapter classes
- [ ] All payment state changes are recorded in `audit_log` — satisfies **REQ-402**
- [ ] JaCoCo branch coverage ≥ 80% for: success → SETTLED; gateway failure → PAYMENT_FAILED + retry scheduled; max retries exceeded → PAYMENT_FAILED permanent — satisfies **NFR-502**
