# S-001 — Shipper Registration

**As a** shipper,
**I want** to register with my business name, contact info, and billing details,
**so that** I can start posting loads on the platform.

## Acceptance Criteria

- [ ] Registration form collects: business name, first name, last name, email, phone number, and billing details.
- [ ] Email must be a valid format and must be unique within the tenant.
- [ ] Password must meet minimum security requirements (e.g., 8+ characters).
- [ ] On successful registration, the user is assigned the SHIPPER role and logged in automatically.
- [ ] On successful registration, the user is redirected to the shipper dashboard.
- [ ] Validation errors are shown inline next to each field without clearing the form.
- [ ] Duplicate email shows a clear error: "An account with this email already exists."
