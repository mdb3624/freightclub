# S-003 — Notification Preferences

**As a** shipper,
**I want** to manage my notification preferences (email, SMS, in-app),
**so that** I only receive alerts through the channels I prefer.

## Acceptance Criteria

- [ ] Profile settings include independent toggles for email, SMS, and in-app notifications.
- [ ] Each toggle can be enabled or disabled independently.
- [ ] Changes are saved immediately on toggle (or on explicit save — must be consistent).
- [ ] Disabling a channel stops that delivery method for all future notifications.
- [ ] At least one channel must remain enabled (or the system gracefully handles all-off state).
- [ ] Preference changes take effect without requiring a logout/login cycle.
