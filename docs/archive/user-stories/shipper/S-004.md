# S-004 — Company Join Code

**As a** shipper,
**I want** to invite colleagues via a join code,
**so that** my team can manage loads under a shared company account.

## Acceptance Criteria

- [ ] A unique join code is generated for each company/tenant at registration time.
- [ ] The join code is visible in the company profile or settings page.
- [ ] A new user who registers using a valid join code is placed in the same tenant.
- [ ] An invalid or expired join code shows a clear error message.
- [ ] All users within the same tenant can view and manage the same set of loads.
- [ ] The join code can be regenerated by an admin user; the old code immediately becomes invalid.
