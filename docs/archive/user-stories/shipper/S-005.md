# S-005 — State Code Dropdown (Origin/Destination)

**As a** shipper,
**I want** origin and destination state captured as a validated 2-letter code via dropdown,
**so that** truckers filtering by state actually see my loads instead of getting zero results.

## Acceptance Criteria

- [ ] Origin state and destination state fields are rendered as dropdowns listing all 50 US states plus DC as 2-letter codes (e.g., "IL — Illinois").
- [ ] Free-text entry is not permitted for state fields.
- [ ] The value stored in the database is exactly 2 characters (e.g., "IL"), never the full state name.
- [ ] Existing loads with full state names (e.g., "Illinois") are migrated to their 2-letter equivalents before this feature ships.
- [ ] Trucker load board filter by origin state returns loads whose `origin_state` matches the selected 2-letter code exactly.
- [ ] Trucker load board filter by destination state returns loads whose `destination_state` matches the selected 2-letter code exactly.
- [ ] Load creation form cannot be submitted with a blank state field.
