# S-011 — Weight Field Contextual Hint

**As a** shipper,
**I want** a contextual hint on the weight field showing "Legal max: 80,000 lbs",
**so that** I don't accidentally post an overweight load that no licensed trucker can legally haul.

## Acceptance Criteria

- [ ] The weight field displays helper text: "Legal max: 80,000 lbs" below or beside the input.
- [ ] The hint is visible without interaction (not hidden behind a tooltip).
- [ ] If the entered weight exceeds 80,000 lbs, a warning is shown: "Exceeds federal weight limit. Confirm this load requires a special permit."
- [ ] The warning does not block submission (it is advisory, not an error), but it must be acknowledged via a checkbox or confirmation before the form can be submitted.
- [ ] Weight field accepts only positive numeric values; non-numeric input is rejected inline.
