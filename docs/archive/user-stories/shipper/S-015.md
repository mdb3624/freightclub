# S-015 — Recurring Load Scheduling

**As a** shipper,
**I want** to schedule a recurring load on a weekly or monthly cadence,
**so that** I don't have to manually re-post the same lane every time.

## Acceptance Criteria

- [ ] Load creation includes an optional "Recurrence" section with options: None, Weekly, Monthly.
- [ ] Weekly recurrence requires a day-of-week selection.
- [ ] Monthly recurrence requires a day-of-month selection (1–28 to handle all months).
- [ ] A recurrence end date (or "Run X times") must be specified.
- [ ] On each recurrence date, the system automatically creates a new DRAFT load pre-filled from the template.
- [ ] The shipper receives an in-app (and optionally email) notification when a recurring draft is created, prompting review and publish.
- [ ] The shipper can cancel the recurrence schedule at any time; existing loads already created are unaffected.
- [ ] Individual recurring load instances can be edited independently without affecting the template.
