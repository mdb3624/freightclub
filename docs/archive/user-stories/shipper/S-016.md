# S-016 — View Trucker Profile Before Claim

**As a** shipper,
**I want** to view a trucker's profile (rating, reviews, equipment, completed loads),
**so that** I can evaluate their reliability before they haul my freight.

## Acceptance Criteria

- [ ] On a claimed load's detail page, the shipper can click the trucker's name to open their public profile.
- [ ] The public profile displays: overall star rating, number of reviews, primary equipment type, and total completed loads.
- [ ] The most recent 5 reviews are displayed with reviewer name, date, and comment.
- [ ] If the trucker has no reviews, the profile shows "No reviews yet."
- [ ] The profile is read-only for the shipper.
- [ ] The trucker's MC/DOT numbers are visible on the profile (already available post-claim per S-040 / existing feature).
