# S-018 — Preferred Carrier List

**As a** shipper,
**I want** a preferred carrier list,
**so that** I can quickly re-use truckers I trust without searching from scratch.

## Acceptance Criteria

- [ ] After a load is completed, the shipper can add the trucker to their preferred carrier list.
- [ ] The preferred carrier list is accessible from the shipper's profile/settings.
- [ ] Each entry shows the trucker's name, rating, equipment type, and date added.
- [ ] The shipper can remove a trucker from the preferred list at any time.
- [ ] When posting a load, the shipper can optionally notify preferred carriers before the load goes public.
- [ ] Preferred carriers are scoped to the shipper's tenant (not globally shared).
