# S-023 — PDF Export per Load

**As a** shipper,
**I want** to export a load's details and documents as a PDF,
**so that** I can file records outside the platform for billing, compliance, or dispute resolution.

## Acceptance Criteria

- [ ] A "Export as PDF" button is available on the load detail page for any load.
- [ ] The exported PDF includes: load reference ID, origin and destination addresses, pickup and delivery windows, commodity, weight, dimensions, equipment type, pay rate, payment terms, and special requirements.
- [ ] The PDF includes the status timeline (if available).
- [ ] The PDF includes embedded or attached BOL and POD images (if uploaded).
- [ ] The PDF is generated server-side and downloaded directly to the browser.
- [ ] The filename follows the pattern: `freightclub-load-{id}-{date}.pdf`.
- [ ] The PDF renders correctly when printed on standard US Letter paper.
