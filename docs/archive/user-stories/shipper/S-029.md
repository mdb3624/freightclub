# S-029 — Email Notification — Load Picked Up

**As a** shipper,
**I want** to receive an email when a trucker marks my load as picked up,
**so that** I know the freight is in transit.

## Acceptance Criteria

- [ ] When a load transitions to IN_TRANSIT status, a `LoadEvent` row is created with `event_type = PICKED_UP`
- [ ] An email is triggered to the shipper's registered email address
- [ ] The email includes load ID, origin, destination, and pickup confirmation timestamp
- [ ] The email subject clearly identifies the action: "Your load [ID] has been picked up"
- [ ] Email is sent within 5 seconds of the status change
- [ ] If email send fails, the status transition still succeeds and an error is logged
