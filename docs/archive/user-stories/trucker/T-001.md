# T-001 — Trucker Registration

**As a** trucker,
**I want** to register with my name, contact info, MC number, and DOT number,
**so that** shippers can verify my credentials before or after I claim their load.

## Acceptance Criteria

- [ ] Registration form collects: first name, last name, email, phone number, MC number, DOT number.
- [ ] MC and DOT number fields accept alphanumeric values; neither is validated against an external registry at this time (format validation only).
- [ ] Email must be a valid format and unique within the tenant.
- [ ] Password must meet minimum security requirements (8+ characters).
- [ ] On successful registration, the user is assigned the TRUCKER role and logged in automatically.
- [ ] On successful registration, the user is redirected to the trucker dashboard.
- [ ] Validation errors are displayed inline without clearing the form.
