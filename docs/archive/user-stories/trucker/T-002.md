# T-002 — Trucker Cost Profile Setup

**As a** trucker,
**I want** to enter my cost profile (fixed costs, fuel $/gal, MPG, maintenance $/mi, monthly miles target, target margin),
**so that** the platform can calculate whether a load is profitable for me.

## Acceptance Criteria

- [ ] Profile settings include a "Cost Profile" section with fields: monthly fixed costs ($), fuel price ($/gal), fuel economy (MPG), maintenance cost ($/mi), monthly miles target, target profit margin (%).
- [ ] All fields accept positive numeric values only.
- [ ] On save, the system calculates and displays: Fixed CPM, Variable CPM, Total CPM, and Minimum RPM (CPM ÷ (1 − margin)).
- [ ] Calculated values update live as the trucker types (or on field blur).
- [ ] Saving the cost profile does not require all fields to be filled — partial profiles are valid (missing fields default to 0).
- [ ] Changes to the cost profile immediately affect RPM badges and profitability breakdowns on the load board and load detail pages.
