# T-005 — CPM and Minimum RPM Display

**As a** trucker,
**I want** my Cost Per Mile (CPM) and minimum RPM calculated and displayed live on my profile,
**so that** I always know my floor rate before I browse loads.

## Acceptance Criteria

- [ ] The profile page displays: Fixed CPM, Variable CPM, Total CPM, and Minimum RPM.
- [ ] Values update live as the trucker edits cost profile fields (or on field blur at minimum).
- [ ] Values are displayed to 2 decimal places with currency symbol (see T-004).
- [ ] If cost profile fields are incomplete, displayed values default to $0.00 for missing inputs rather than throwing an error.
- [ ] Minimum RPM calculation: Total CPM ÷ (1 − target margin %). If margin is 0%, Minimum RPM equals Total CPM.
