# T-008 — Load Board Sorting

**As a** trucker,
**I want** to sort the load board by RPM, distance, and pickup date,
**so that** I can quickly identify the most profitable or most urgent loads without manual comparison.

## Acceptance Criteria

- [ ] Load board table headers for RPM, Distance, and Pickup Date are clickable to sort.
- [ ] Clicking a header sorts ascending; clicking again sorts descending; a third click resets to default order.
- [ ] The active sort column is visually indicated with an arrow icon (↑ or ↓).
- [ ] Default sort is by pickup date ascending (soonest pickup first).
- [ ] Sort state is preserved in URL query params (see P-002) so it survives navigation.
- [ ] Sorting applies to the current filtered result set, not the full unfiltered list.
- [ ] Loads with null values for the sort column are placed at the end of the list.
