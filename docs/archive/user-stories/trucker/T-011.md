# T-011 — Pickup Date Urgency Signal

**As a** trucker,
**I want** loads picking up within 24 hours highlighted with an urgency signal,
**so that** I don't miss time-sensitive opportunities that disappear before I notice them.

## Acceptance Criteria

- [ ] Loads with an earliest pickup date within 24 hours from now display an urgency badge (e.g., "Picks up today" or "Picks up in 6 hrs").
- [ ] Loads with an earliest pickup date between 24 and 48 hours display a secondary signal (e.g., "Picks up tomorrow") in amber.
- [ ] Loads within 24 hours use a more prominent red or amber highlight on the row or card.
- [ ] The urgency signal is based on `pickup_from` (earliest pickup) relative to the viewer's current local time.
- [ ] The urgency signal updates dynamically — a load that crosses the 24-hour threshold while the page is open refreshes its state (on next data fetch, at minimum).
- [ ] Loads past their pickup window but not yet claimed are visually flagged as "Overdue" to alert truckers and prompt shipper action.
