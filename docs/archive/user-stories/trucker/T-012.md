# T-012 — Profitability Card on Claimed Load

**As a** trucker,
**I want** to see the profitability breakdown card on a load I've already claimed,
**so that** I can reference cost analysis during execution, not just during the decision phase.

## Acceptance Criteria

- [ ] The profitability breakdown card (estimated revenue, fuel cost, maintenance cost, net profit, RPM vs minimum RPM) is visible on the load detail page for loads in CLAIMED and IN_TRANSIT status.
- [ ] The card shows the same calculations as when the load was OPEN.
- [ ] If the trucker's cost profile changes after claiming, the card reflects the updated calculations.
- [ ] The card is hidden or replaced with a summary line only after the load reaches DELIVERED or SETTLED status (where actuals may differ).
- [ ] The card is not visible to the shipper — it is trucker-only.
