# T-014 — Suggested Loads Based on Preferred Lanes

**As a** trucker,
**I want** suggested loads based on my preferred lanes,
**so that** I spend less time searching and more time driving profitable routes.

## Acceptance Criteria

- [ ] The trucker can configure preferred lanes in their profile: one or more origin state + destination state pairs.
- [ ] The trucker dashboard displays a "Suggested Loads" section showing open loads matching any of their preferred lanes.
- [ ] Suggested loads are sorted by pickup date ascending (soonest first).
- [ ] The section is hidden if no preferred lanes are configured, with a prompt to set them up.
- [ ] Suggested loads link to the full load detail page.
- [ ] Suggestions update on each page load (not cached longer than 5 minutes).
- [ ] A trucker can set up to 10 preferred lane pairs.
