# T-017 — HOS Widget: 70-Hour/8-Day Cycle (Load Execution View)

**As a** trucker,
**I want** the HOS widget to show my 70-hour/8-day rolling on-duty total alongside my per-shift limits,
**so that** I comply with FMCSA multi-day cycle rules and can plan stops before I run out of hours.

## Acceptance Criteria

- [ ] The HOS widget on the trucker dashboard and load execution page displays three progress elements: 11-hr drive limit, 14-hr on-duty window, and 70-hr/8-day cycle.
- [ ] The 70-hr cycle bar shows hours used / 70 hours and hours remaining.
- [ ] The 8-day breakdown (hours per day for the last 8 days) is accessible from the widget (e.g., expandable section).
- [ ] Color coding: green (>14 hr remaining), amber (≤14 hr remaining), red (≤2 hr remaining), and gray/striped (limit reached — must stop).
- [ ] When the 70-hr limit is reached, a prominent warning is shown: "You have exhausted your 70-hour cycle. You must take a 34-hour restart before driving."
- [ ] This story and T-007 share the same underlying feature; both must be satisfied by the same implementation.

## Dependencies
