# T-018 — HOS On-Duty Bar Before Start Time Entered

**As a** trucker,
**I want** the HOS on-duty bar to render and prompt me to enter my start time even before I've entered it,
**so that** I understand what data is missing and the widget isn't just silently empty.

## Acceptance Criteria

- [ ] When no shift start time has been entered, the HOS widget renders the on-duty bar in an empty/inactive state with a clear inline prompt: "Enter your shift start time to track hours."
- [ ] A start time input is accessible directly within or adjacent to the widget — the trucker doesn't have to navigate away.
- [ ] The drive and on-duty bars show "—" or "Not started" labels, not 0:00, to distinguish "not started" from "zero hours used."
- [ ] Once the start time is entered, bars immediately begin calculating and displaying time elapsed.
- [ ] The start time field is pre-filled with the current time as a default, which the trucker can adjust.
