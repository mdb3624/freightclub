# T-021 — Report Issue During Transit

**As a** trucker,
**I want** to report an issue during transit (delay, damage, accident),
**so that** shippers and dispatchers are informed in real time and I have a documented record.

## Acceptance Criteria

- [ ] A "Report Issue" action is available on the load detail page when the load is IN_TRANSIT.
- [ ] Issue types include: Delay, Cargo Damage, Vehicle Breakdown, Accident, Other.
- [ ] A required description field (minimum 20 characters) captures the details.
- [ ] Optionally, the trucker can upload a photo of the issue.
- [ ] On submission, the shipper receives an in-app and email notification with the issue type, description, and photo (if any).
- [ ] The issue is logged in the load's status timeline (see S-021).
- [ ] Multiple issues can be reported on a single load.
- [ ] Reported issues do not change the load status — they are informational records only.
