## User Story

As a **trucker**, I need to see current diesel prices by region so that I can estimate fuel costs and make informed decisions about which loads are profitable.

## Business Rules

- EIA API endpoint: `https://www.eia.gov/opendata/qb.php?category=PET.EW...` (public API, no authentication).
- Cache TTL: 6 hours (prevents hammering EIA API).
- Regions supported: PADD1 (East Coast), PADD2 (Midwest), PADD3 (Gulf Coast), PADD4 (Rocky Mountain), PADD5 (West Coast).
- Fallback: If API fails, show last cached price with 3-hour staleness warning.
- Data freshness: Prices updated weekly by EIA (typically Thursday).

---

## Implementation Notes

- **Backend:** `MarketDataService` fetches from EIA; `@Cacheable("dieselPrices", 6 hours)` decorator handles TTL.
- **Endpoint:** GET `/api/v1/market/loads` (returns load + price data) or separate GET `/api/v1/market/fuel-prices/{region}`.
- **Integration:** HTTP client (RestTemplate or WebClient) with timeout (10 seconds) and retry (2 attempts).
- **Error Handling:** 500 error → fallback to cached price; 429 (rate limit) → retry after 1 hour.
- **Frontend:** Load price widget in `EarningSummaryCard.tsx` or dedicated `FuelPriceWidget.tsx` component.
- **Tests:** `MarketDataServiceTest.java` mocks EIA response; frontend tests verify widget rendering with cached data.
