## User Story

**As a** *Shipper*,  
**I want to** *view detailed public performance metrics for any carrier*,  
**so that** *I can make informed decisions about carrier reliability, quality, and fit for my loads.*

---

## Acceptance Criteria

### AC-1: Shipper Views Carrier Performance Profile

**Given** a shipper is viewing a carrier  
**When** they click "View Profile" or "Performance"  
**Then** display public metrics:
- Acceptance rate (% of loads accepted vs. offered)
- On-time delivery rate (% delivered by promised date)
- Average delivery time (hours from pickup to delivery)
- Quality score (1-100 based on damage/theft/compliance)
- Number of shippers who have marked as "preferred" (social proof)
- Operating region (service areas)
- Equipment types available
- Years in business

---

### AC-2: Benchmark Comparison

**Given** shipper is viewing a carrier profile  
**When** carrier metrics are displayed  
**Then** show:
- Carrier's metric
- Platform average (all carriers for that tenant)
- Percentile ranking (e.g., "Top 15% for on-time rate")
- Visual indicator (green if above avg, yellow if equal, red if below)

---

### AC-3: Carrier Interest Count

**Given** carrier profile is viewed  
**When** profile is displayed  
**Then** show:
- "Viewed by X shippers in last 30 days"
- "X shippers have preferred this carrier"
- Social proof indicator (trending up/down)

---

### AC-4: Multi-Tenancy

**Given** carrier profile is requested  
**When** shipper views profile  
**Then** MUST:
- Show only carrier metrics for requesting shipper's tenant
- Hide competitor's metrics
- Enforce isolation: metrics aggregated per tenant

---

## Technical Context

### API Endpoints
- GET `/api/v1/carriers/{carrierId}/performance` → Individual profile
- GET `/api/v1/analytics/top-carriers` → Ranked list by quality
- GET `/api/v1/analytics/carrier-benchmarks` → Tenant-wide benchmarks

### Data Source
```
CarrierPerformance
- carrierId, tenantId
- acceptanceRate, onTimeRate, avgDeliveryTimeHours, qualityScore
- preferredByCount, loadsAccepted, loadsCompleted
- updatedAt (recalculated hourly)
```
