**Status:** READY_FOR_DESIGN

## User Story

**As an** *Admin or Shipper*,  
**I want to** *see how many carriers are viewing my loads and how many shippers are viewing carrier profiles*,  
**so that** *I can understand market demand, optimize pricing, and improve engagement.*

---

## Acceptance Criteria

### AC-1: Track Load Views by Carriers

**Given** a carrier views a load  
**When** they open the load details  
**Then** system:
- Records the view event (carrierId, loadId, timestamp)
- Counts total views per load
- Counts unique carriers who viewed the load

---

### AC-2: Shipper Sees Load View Metrics

**Given** shipper is viewing their load  
**When** they click "View Analytics" or "Interest"  
**Then** display:
- "Viewed by X carriers in the last 7/30 days"
- "X unique carriers interested"
- View trend (up/down arrow)
- Correlation with claim rate (if viewed X times, claimed Y times)

---

### AC-3: Track Carrier Profile Views

**Given** a shipper views a carrier profile  
**When** they open the carrier's details  
**Then** system:
- Records view event (shipperId, carrierId, timestamp)
- Counts total views on carrier profile
- Counts unique shippers who viewed carrier

---

### AC-4: Admin Views Engagement Metrics

**Given** admin is in Analytics dashboard  
**When** they navigate to "Engagement" section  
**Then** display:
- Total load views (platform-wide)
- Average views per load
- Most viewed loads (top 10)
- Carrier profile view trends
- Engagement conversion (views → claims/interactions)

---

### AC-5: Real-Time View Recording

**Given** carrier/shipper navigates  
**When** they open a profile/load details  
**Then** MUST:
- Record view asynchronously (non-blocking)
- Update view counts within 5 seconds
- Never fail load rendering if tracking fails

---

### AC-6: Multi-Tenancy

**Given** view events are recorded  
**When** tracked  
**Then** MUST:
- Associate with correct tenant_id
- Isolation: loads only count views from same tenant
- Analytics aggregated per tenant scope

---

## Technical Context

### Domain Entities
```
LoadView
- id (UUID)
- loadId (UUID)
- carrierId (UUID)
- tenantId (UUID)
- viewedAt (TIMESTAMPTZ)
- createdAt (TIMESTAMPTZ)

CarrierProfileView
- id (UUID)
- carrierId (UUID)
- shipperId (UUID)
- tenantId (UUID)
- viewedAt (TIMESTAMPTZ)
- createdAt (TIMESTAMPTZ)
```

### Queries
- `COUNT(LoadView) WHERE loadId = X AND tenantId = Y`
- `COUNT(DISTINCT carrierId) FROM LoadView WHERE loadId = X` → Interest
- `COUNT(CarrierProfileView) WHERE carrierId = X AND tenantId = Y`
- `COUNT(DISTINCT shipperId) FROM CarrierProfileView WHERE carrierId = X` → Interest

### Frontend Hook Calls
- After opening load: `recordLoadView(loadId, carrierId)`
- After opening carrier profile: `recordProfileView(carrierId, shipperId)`
- Query: `getLoadViewCount(loadId)`, `getLoadInterest(loadId)`
- Query: `getCarrierProfileViewCount(carrierId)`, `getCarrierInterest(carrierId)`
