# US-751: Code-Split Auth Module from Dashboard Bundle

**Phase:** Infrastructure (Performance Optimization)  
**Status:** DESIGN_APPROVED  
**Priority:** CRITICAL  
**Effort:** 5-8 hours  
**Impact:** 15% reduction in login page bundle size (76 KB savings)

---

## User Story

As a **platform operator**, I need to split the authentication module (login/register) into a separate code bundle so that users accessing the login page don't download unnecessary dashboard code, enabling faster initial page load and reduced Time to Interactive (TTI).

---

## Acceptance Criteria

| ID | Criterion | Status |
|---|---|---|

---

## Business Rules

- Auth module must include: LoginPage, RegisterPage, auth hooks, auth store
- Dashboard module must include: TruckerDashboard, ShipperDashboard, load features
- Shared vendor code must not duplicate across chunks
- React Query must load only after successful authentication
- Code-splitting must be transparent to user experience (no delays in navigation)
- All existing routes must remain functional after split
