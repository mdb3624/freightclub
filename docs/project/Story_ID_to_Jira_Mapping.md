
## Story ID Format

- **US-### :** User stories (features, functionality)
- **US-###-v# :** Story variants/versions (e.g., US-103-v2, US-707-v2)
- **CHG-### :** Change requests (issues discovered mid-implementation)
- **SEC-### :** Security stories
- **INF-### :** Infrastructure stories
