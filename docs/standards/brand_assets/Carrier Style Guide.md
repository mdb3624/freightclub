# Carrier Style Guide.md

This style guide for the mobile carrier dashboard is defined by a **"Luxury Industrial" dark-mode aesthetic**. It prioritizes high-contrast visibility and a premium, tactile feel, suitable for professional use in low-light and high-glare road conditions.

### 1. Color Palette: "Dark & Metallic"
* **Primary Background:** Deep charcoal/black (`#121212`), which provides a high-contrast canvas that reduces eye strain.
* **Surface/Accent Colors:** A rich, metallic copper/bronze gradient. This is used for cards, buttons, and text highlights, creating a "premium" feel that distinguishes the brand from standard utility dashboards.
* **Text Colors:** Soft off-white for primary text (readability) and a muted metallic copper for secondary labels and icons to maintain the cohesive color scheme.

### 2. Typography
* **Style:** Clean, sans-serif typeface with a geometric, modern structure.
* **Hierarchy:** 
  * **KPIs (e.g., 97%, $16.3B):** Extra-bold, large, and high-contrast to emphasize "at-a-glance" data.
  * **Section Headers:** Uppercase, spaced-out (kerning) lettering to create a sophisticated, wide-set look.
  * **Body/Inputs:** Clear, medium-weight font to ensure readability against the dark background.

### 3. Layout & Structure
* **Vertical Stacked Cards:** The design utilizes a card-based system where distinct modules (Active Loads, Search Module, Navigation) are contained within rounded, border-highlighted rectangles.
* **Integrated Components:** The layout is optimized for a phone screen by stacking modules vertically, ensuring the user can access core functions—like searching for loads or checking payments—with a single thumb.
* **Rounded Geometry:** Heavy use of "pill-shaped" (fully rounded) buttons and cards, which makes the interface feel modern and easier to tap.

### 4. Interface Elements
* **Depth & Lighting:** The design uses subtle internal shadows and glow effects on the borders to make cards appear "raised" off the dark background.
* **Touch Targets:** Buttons like "Search Loads" and "Expand Map" are wide, full-width, or large-format to ensure they are easy to hit on a mobile device.
* **Iconography:** Minimalist, thin-line icons. They are kept sparse to prevent clutter, allowing the data numbers to be the primary focus.

### 5. Brand Assets: Logo & Favicon
* **Primary System Logo (`web_logo.png`):** 
  * **Visual Graphic:** A high-detail semi-truck cab facing right, featuring a polished metallic copper/bronze gradient finish with distinct lighting highlights. The truck has trailing aerodynamic horizontal speed lines and a dust/smoke cloud effect at the rear wheels to imply forward kinetic motion.
  * **Primary Typography:** The uppercase text **"MDB"** is rendered in a bold, extra-large geometric sans-serif typeface with a dramatic 3D extrusion effect, deep inner shading, and a drop shadow that matches the metallic copper/bronze scheme.
  * **Sub-Branding Typography:** Positioned directly beneath the primary text is the label **"Integrated Logistics"** in a clean, medium-weight, high-legibility sans-serif font using a solid, uniform copper color.
* **System Favicon (`web_logo_favicon.png`):**
  * **Visual Asset:** A scaled, aspect-ratio-locked downsized variant of `web_logo.png`. It maintains the full visual layout—including the kinetic semi-truck icon, the 3D **"MDB"** lettering, and the **"Integrated Logistics"** sub-text—optimized for browser tabs and application shortcut icons.
