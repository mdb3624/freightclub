# INF-002 — Spring Boot Actuator: `/health` and `/info`

**Phase:** 1.2
**Persona:** Platform
**Status:** ✅ Built

## Story

As an operator, I want a `/actuator/health` endpoint so that load balancers, uptime monitors, and the `/start` dev script can verify the backend is running without triggering authentication.

## Acceptance Criteria

- `spring-boot-starter-actuator` dependency is present in `pom.xml`
- `GET /actuator/health` returns `{"status": "UP"}` with HTTP 200 when the app is healthy
- `GET /actuator/info` returns application version and build info
- `/actuator/health` and `/actuator/info` are accessible without a JWT (permitted in `SecurityConfig`)
- All other actuator endpoints remain secured or disabled
