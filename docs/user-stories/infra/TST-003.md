# TST-003 — `JwtAuthenticationFilter` Unit Tests

**Phase:** 1.2
**Persona:** Platform
**Status:** ✅ Built

## Story

As a developer, I want `JwtAuthenticationFilter` covered by unit tests so that token extraction, validation, and rejection are verified across the full range of token states.

## Acceptance Criteria

- Valid token: sets `SecurityContext` with correct user details and tenant ID
- Missing `Authorization` header: passes request through unauthenticated (permits public endpoints)
- Malformed token (not a JWT): returns 401
- Expired token: returns 401
- Wrong issuer (`iss` mismatch): returns 401
- Wrong audience (`aud` mismatch): returns 401
- Tampered signature: returns 401
- All test cases use `@Mock` — no full Spring Security context loaded
