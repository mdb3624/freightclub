# S-009 — Pickup/Delivery Window Label Rename

**As a** shipper,
**I want** pickup and delivery window fields labeled "Earliest Pickup / Latest Pickup / Earliest Delivery / Latest Delivery",
**so that** I'm never confused about whether "from" refers to a location or a point in time.

## Acceptance Criteria

- [ ] The pickup window start field is labeled "Earliest Pickup" (not "Pickup From").
- [ ] The pickup window end field is labeled "Latest Pickup" (not "Pickup To").
- [ ] The delivery window start field is labeled "Earliest Delivery" (not "Delivery From").
- [ ] The delivery window end field is labeled "Latest Delivery" (not "Delivery To").
- [ ] The same labels appear on: load creation form, load edit form, load detail view (shipper), and load detail view (trucker).
- [ ] No underlying field names or API payloads change — label only.
