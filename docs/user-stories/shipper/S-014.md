# S-014 — Duplicate a Load

**As a** shipper,
**I want** to duplicate an existing load,
**so that** I can quickly re-post recurring lanes without re-entering all details.

## Acceptance Criteria

- [ ] A "Duplicate" action is available on any load regardless of status.
- [ ] Duplicating opens the load creation form pre-filled with all fields from the source load.
- [ ] The duplicate starts in DRAFT status regardless of the source load's status.
- [ ] Date/time fields (pickup window, delivery window) are cleared and must be re-entered.
- [ ] The trucker assignment (`trucker_id`) is not copied.
- [ ] The shipper can edit any pre-filled field before saving or publishing.
- [ ] Saving the duplicate creates a new, independent load record with its own ID.
