# S-017 — Accept or Reject Trucker Bids

**As a** shipper,
**I want** to accept or reject bids from truckers,
**so that** I control who hauls my freight rather than operating purely first-come-first-served.

## Acceptance Criteria

- [ ] A load can be posted in "Open to Bids" mode (distinct from first-come-first-served).
- [ ] Truckers can submit a bid (with optional counter-rate) on a bid-mode load.
- [ ] The shipper receives an in-app notification for each new bid.
- [ ] The shipper's load detail page lists all bids with trucker name, rating, and offered rate.
- [ ] The shipper can accept one bid; doing so assigns the trucker and transitions the load to CLAIMED.
- [ ] All other open bids are automatically rejected and those truckers are notified.
- [ ] The shipper can reject individual bids with an optional reason.
- [ ] A rejected trucker is notified of the rejection.
- [ ] Bids can only be submitted while the load is in OPEN status.
