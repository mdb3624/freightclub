# S-022 — View BOL and POD Documents

**As a** shipper,
**I want** to view a trucker-uploaded signed BOL photo and Proof of Delivery photo on the load detail page,
**so that** I have delivery confirmation in one place without chasing down paperwork.

## Acceptance Criteria

- [ ] The load detail page has a "Documents" section visible to the shipper.
- [ ] When a trucker uploads a BOL photo at pickup, it appears in this section labeled "Bill of Lading."
- [ ] When a trucker uploads a POD photo at delivery, it appears labeled "Proof of Delivery."
- [ ] Documents can be previewed inline (image viewer) without leaving the page.
- [ ] Documents can be downloaded individually.
- [ ] If no document has been uploaded yet, a placeholder reads "Not yet uploaded."
- [ ] Only the assigned trucker can upload; the shipper can only view.
