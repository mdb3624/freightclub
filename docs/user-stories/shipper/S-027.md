# S-027 — Shipper Public Reputation Profile

**As a** shipper,
**I want** my public profile to show my overall rating, average payment speed, completed load count, and dispute/cancellation flags,
**so that** quality truckers are willing to haul my loads.

## Acceptance Criteria

- [ ] Each shipper has a public profile page viewable by truckers.
- [ ] Profile displays: overall star rating (average of all trucker reviews), number of reviews, total completed loads, average payment speed (days from delivery to payment), and any active dispute or cancellation flags.
- [ ] A flag is shown if the shipper has cancelled more than 10% of claimed loads in the past 90 days.
- [ ] A flag is shown if the shipper has active unresolved payment disputes.
- [ ] Truckers can access the shipper's public profile from the load board card and from the load detail page.
- [ ] The shipper can view their own profile as truckers see it from their account settings.
- [ ] Shipper can see their full rating history (all reviews received) from their private settings page.
