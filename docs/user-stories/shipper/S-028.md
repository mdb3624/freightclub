# S-028 — Email Notification — Load Claimed

**As a** shipper,
**I want** to receive an email when a trucker claims my load,
**so that** I know freight is moving and can prepare for pickup coordination.

## Acceptance Criteria

- [ ] When a load transitions to CLAIMED status, a `LoadEvent` row is created with `event_type = CLAIMED`
- [ ] An email is triggered to the shipper's registered email address
- [ ] The email includes load ID, origin address, destination address, and the trucker's name and contact info
- [ ] The email subject clearly identifies the action: "Your load [ID] has been claimed"
- [ ] Email is sent synchronously or asynchronously but within 5 seconds of the claim
- [ ] If email send fails, the claim still succeeds and an error is logged (no user-blocking failure)
