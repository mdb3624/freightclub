# T-003 — Cost Profile Setup CTA

**As a** trucker,
**I want** a prominent call-to-action to set up my cost profile when none is configured,
**so that** I understand why RPM badges and profitability breakdowns are empty and know what to do about it.

## Acceptance Criteria

- [ ] When a trucker has no cost profile configured, a prominent banner or card is displayed on the dashboard and load board (not a footnote or small hint text).
- [ ] The CTA reads something like: "Set up your cost profile to see profitability ratings on loads."
- [ ] Clicking the CTA navigates directly to the cost profile section of the trucker's profile settings.
- [ ] The CTA is dismissed automatically once a cost profile is saved (at least one field populated).
- [ ] RPM badges on the load board show a neutral state (e.g., "—" or "Set up profile") rather than a blank or misleading value when no profile exists.
- [ ] The profitability breakdown card on load detail shows a prompt rather than empty fields when no profile exists.
