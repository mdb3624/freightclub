# T-004 — RPM Display Precision

**As a** trucker,
**I want** RPM values displayed to 2 decimal places,
**so that** the numbers are readable and don't show unnecessary precision that adds cognitive noise.

## Acceptance Criteria

- [ ] All RPM values displayed in the UI are rounded to exactly 2 decimal places (e.g., "$2.34/mi", not "$2.3412/mi").
- [ ] This applies to: RPM column on load board, RPM badge on load cards, profitability breakdown card, minimum RPM on profile.
- [ ] Rounding uses standard half-up rounding.
- [ ] The underlying stored/calculated value retains full precision; only the display is rounded.
- [ ] Currency symbol and "/mi" unit label are always present alongside the value.
