# T-006 — 30-Day Earnings Summary

**As a** trucker,
**I want** a 30-day earnings summary showing loads completed, total miles, total revenue, and effective CPM,
**so that** I can track whether I'm hitting my financial targets.

## Acceptance Criteria

- [ ] The trucker dashboard or history tab displays a 30-day earnings summary section.
- [ ] Summary includes: number of loads completed, total miles driven (sum of load distances), total revenue earned, and effective CPM (total revenue ÷ total miles).
- [ ] The 30-day window is a rolling window (last 30 days from today), not a calendar month.
- [ ] Only loads with status DELIVERED or SETTLED within the window are counted.
- [ ] Effective CPM is compared to the trucker's target CPM with a visual indicator (e.g., green if above target, red if below).
- [ ] If no loads were completed in the window, the summary shows zeros with a prompt to start finding loads.
