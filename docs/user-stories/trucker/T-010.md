# T-010 — Equipment Filter Unlockable

**As a** trucker,
**I want** to unlock the equipment type filter even when my profile has a default type set,
**so that** I can browse loads for other trailer types I own without having to change my profile.

## Acceptance Criteria

- [ ] When a trucker's profile has a default equipment type, the equipment filter on the load board defaults to that type but is NOT locked — it is an editable dropdown.
- [ ] The trucker can change the equipment filter to any available type at any time.
- [ ] Changing the equipment filter does not modify the trucker's saved profile equipment type.
- [ ] An optional "Reset to my equipment" link appears when the filter diverges from the profile default.
- [ ] The selected equipment filter is preserved in URL query params (see T-009).
