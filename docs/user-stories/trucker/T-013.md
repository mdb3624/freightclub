# T-013 — Shipper Reputation Badge on Load Cards

**As a** trucker,
**I want** to see a shipper's reputation badge (star rating, average payment speed) on load board cards,
**so that** I can avoid slow-paying or unreliable shippers before I commit to a load.

## Acceptance Criteria

- [ ] Each load card/row on the load board displays the posting shipper's overall star rating (e.g., "★ 4.2").
- [ ] Average payment speed is displayed in days (e.g., "Pays in ~12 days").
- [ ] If the shipper has fewer than 3 reviews, the rating is shown as "New" instead of a numeric score.
- [ ] Clicking the badge opens the shipper's public profile (see S-027).
- [ ] A dispute or high-cancellation flag on the shipper is surfaced with a warning icon on the badge.
- [ ] Badge data is loaded with the load list — no separate per-load request required.
