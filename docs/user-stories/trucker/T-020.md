# T-020 — BOL and POD Photo Upload

**As a** trucker,
**I want** to upload a BOL photo at pickup and a Proof of Delivery photo at delivery,
**so that** the shipper and platform have confirmation at each stage and disputes are supported by evidence.

## Acceptance Criteria

- [ ] After marking a load as picked up (IN_TRANSIT), the trucker can upload a BOL photo from the load detail page.
- [ ] After marking a load as delivered (DELIVERED), the trucker can upload a POD photo.
- [ ] Accepted file types: JPEG, PNG, HEIC, PDF. Max file size: 10 MB per upload.
- [ ] Upload includes a progress indicator; success is confirmed with a thumbnail preview.
- [ ] Uploaded documents are immediately visible to the shipper on the load detail page (see S-022).
- [ ] A trucker can replace a previously uploaded document (e.g., if the photo was blurry) as long as the load has not reached SETTLED status.
- [ ] Failed uploads show a clear error with a retry option.
