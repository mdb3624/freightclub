# T-025 — Active Load Slot Freed on Shipper Cancellation

**As a** trucker,
**I want** my active load slot to be freed immediately when a shipper cancels a claimed load,
**so that** I can claim a new load without delay.

## Acceptance Criteria

- [ ] When a shipper cancels a CLAIMED load, the load's `status` transitions to CANCELLED
- [ ] The load's `trucker_id` is cleared (set to NULL) to release the trucker's active slot
- [ ] The active `Claim` row's `status` transitions to CANCELLED and `released_at` is set
- [ ] The trucker can immediately claim a new load after cancellation (one active load per trucker limit is no longer blocking)
- [ ] The trucker's dashboard immediately reflects the freed slot (no refresh required)
- [ ] The cancellation is logged in `LoadEvent` with `event_type = CANCELLED`
