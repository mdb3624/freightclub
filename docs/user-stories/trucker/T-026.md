# T-026 — Live Diesel Price Market Ticker

**As a** trucker,
**I want** to see live regional diesel prices (Diesel West, Diesel South) in a market ticker,
**so that** I have current fuel cost data when evaluating load profitability.

## Acceptance Criteria

- [ ] A market ticker widget is displayed on the load board and/or dashboard
- [ ] The ticker shows two prices: "DIESEL WEST" and "DIESEL SOUTH" in format `$X.XX/gal`
- [ ] Prices are sourced from the EIA (U.S. Energy Information Administration) API via backend proxy (P-005)
- [ ] A week-over-week delta indicator (e.g. "+$0.15" or "-$0.08") is shown for each price
- [ ] The delta is color-coded: red for rising, green for falling
- [ ] A timestamp or "last updated" indicator shows when the data was last refreshed
- [ ] If prices are > 48 hours old, a stale data warning is displayed (e.g. "Last updated 2 days ago")
- [ ] Clicking the ticker or "data sources" section reveals EIA attribution: "Data: U.S. EIA"
- [ ] Ticker updates every 6 hours (or when user manually refreshes)
