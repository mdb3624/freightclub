Design Features:

  • Header: Title + "Track Shipments" action button (bronze gradient)
  • Search: Real-time filter by Load ID, destination, or carrier name
  • Status Badges: Color-coded (Red=Urgent, Blue=OK, Green=Delivered)
  • Carrier Rating: Star display (★★★★☆) for quick quality assessment
  • Grid Layout: 6-column responsive grid, sticky header, hover states
  • Typography: Georgia serif for headings, clean sans-serif for data
  • Color Palette: White bg, bronze accent (#b08d57), status colors
  • Refinement: Generous whitespace, subtle shadows, professional tone
